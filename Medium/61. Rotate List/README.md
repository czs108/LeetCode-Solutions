# 61. Rotate List

> Medium

------

Given a linked list, rotate the list to the right by `k` places, where `k` is non-negative.

**Example 1:**

![list-1](images/list-1.jpg)

```
Input: head = [1,2,3,4,5], k = 2
Output: [4,5,1,2,3]
```

**Example 2:**

![list-2](images/list-2.jpg)

```
Input: head = [0,1,2], k = 4
Output: [2,0,1]
```
