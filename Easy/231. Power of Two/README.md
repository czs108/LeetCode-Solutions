# 231. Power of Two

> Easy

------

Given an integer `n`, return `true` if it is a power of two. Otherwise, return `false`.

**Example 1:**

```
Input: n = 1
Output: true
Explanation: 2 ^ 0 = 1
```

**Example 2:**

```
Input: n = 16
Output: true
Explanation: 2 ^ 4 = 16
```

**Example 3:**

```
Input: n = 3
Output: false
```

**Example 4:**

```
Input: n = 4
Output: true
```

**Example 5:**

```
Input: n = 5
Output: false
```
