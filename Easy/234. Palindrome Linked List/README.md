# 234. Palindrome Linked List

> Easy

------

Given the `head` of a singly linked list, return `true` if it is a palindrome.

**Example 1:**

![list-1](images/list-1.jpg)

```
Input: head = [1,2,2,1]
Output: true
```

**Example 2:**

![list-2](images/list-2.jpg)

```
Input: head = [1,2]
Output: false
```
