# 206. Reverse Linked List

> Easy

------

Given the `head` of a singly linked list, reverse the list, and return the reversed list.

**Example 1:**

![list-1](images/list-1.jpg)

```
Input: head = [1,2,3,4,5]
Output: [5,4,3,2,1]
```

**Example 2:**

![list-2](images/list-2.jpg)

```
Input: head = [1,2]
Output: [2,1]
```

**Example 3:**

```
Input: head = []
Output: []
```
